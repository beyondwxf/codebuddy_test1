## 项目概述

CodeBuddy任务管理系统是一个基于RuoYi风格的前后端分离后台管理系统，采用Spring Boot 3.2.0 + Java 17 + Vue 3技术栈构建。项目包含完整的权限管理体系、商品WMS系统和电商模块。

## 系统要求

### 硬件要求
- **内存**: 至少 8GB RAM（推荐 16GB+）
- **存储**: 至少 5GB 可用空间
- **处理器**: Intel i5 或同等性能的AMD处理器

### 软件要求
- **JDK**: Java 17
- **数据库**: MySQL 8.0+
- **缓存**: Redis 6.0+
- **构建工具**: Maven 3.6+
- **前端**: Node.js 16+，npm 8+

## JDK 17 安装配置

### Windows 系统安装步骤

1. **下载 JDK 17**
   - 访问 Oracle 官网或 OpenJDK 官网
   - 下载 Windows x64 版本的 JDK 17

2. **安装 JDK 17**
   ```cmd
   # 运行下载的安装程序
   # 默认安装路径: C:\Program Files\Java\jdk-17
   ```

3. **设置环境变量**
   ```cmd
   # 设置 JAVA_HOME
   setx JAVA_HOME "C:\Program Files\Java\jdk-17"
   
   # 添加到 PATH
   setx PATH "%PATH%;%JAVA_HOME%\bin"
   ```

4. **验证安装**
   ```cmd
   java -version
   javac -version
   ```

### Linux/Mac 系统安装步骤

1. **使用包管理器安装**
   ```bash
   # Ubuntu/Debian
   sudo apt update
   sudo apt install openjdk-17-jdk
   
   # CentOS/RHEL/Fedora
   sudo yum install java-17-openjdk-devel
   
   # macOS (使用 Homebrew)
   brew install openjdk@17
   ```

2. **配置环境变量**
   ```bash
   # 添加到 ~/.bashrc 或 ~/.zshrc
   export JAVA_HOME=/usr/lib/jvm/java-17-openjdk
   export PATH=$PATH:$JAVA_HOME/bin
   
   # 重新加载配置
   source ~/.bashrc
   ```

## MySQL 数据库安装配置

### 安装步骤

1. **下载 MySQL**
   - 访问 MySQL 官网下载社区版
   - 选择 Windows x86_64 版本

2. **安装 MySQL**
   ```cmd
   # 运行安装程序
   # 选择 "Server only" 或 "Developer Default"
   # 设置 root 用户密码
   ```

3. **配置 MySQL**
   ```sql
   -- 登录 MySQL
   mysql -u root -p
   
   -- 创建数据库
   CREATE DATABASE IF NOT EXISTS ruoyi_admin 
   CHARACTER SET utf8mb4 
   COLLATE utf8mb4_general_ci;
   
   -- 创建用户
   CREATE USER 'ruoyi'@'%' IDENTIFIED BY 'Ruoyi@2024';
   GRANT ALL PRIVILEGES ON ruoyi_admin.* TO 'ruoyi'@'%';
   FLUSH PRIVILEGES;
   ```

### 配置文件优化

1. **my.ini 配置**
   ```ini
   [mysqld]
   # 字符集设置
   character-set-server=utf8mb4
   collation-server=utf8mb4_general_ci
   
   # 连接数设置
   max_connections=200
   innodb_buffer_pool_size=256M
   
   # 日志设置
   log-error=.err
   slow_query_log=1
   long_query_time=2
   ```

2. **验证配置**
   ```sql
   SHOW VARIABLES LIKE 'character_set%';
   SHOW VARIABLES LIKE 'innodb_buffer_pool_size';
   ```

## Redis 安装启动

### Windows 安装步骤

1. **下载 Redis**
   - 访问 GitHub Releases 页面
   - 下载 Windows 版本的 Redis

2. **安装 Redis**
   ```cmd
   # 解压到 C:\redis\
   # 创建配置文件 redis.windows.conf
   ```

3. **配置 Redis**
   ```conf
   # redis.windows.conf
   port 6379
   bind 127.0.0.1
   requirepass your_redis_password
   maxmemory 256mb
   maxmemory-policy allkeys-lru
   ```

4. **启动 Redis**
   ```cmd
   # 以服务方式启动
   redis-server.exe redis.windows.conf
   
   # 或以命令行启动
   redis-server.exe --service-start
   ```

### Linux 安装步骤

1. **使用包管理器安装**
   ```bash
   # Ubuntu/Debian
   sudo apt install redis-server
   
   # CentOS/RHEL/Fedora
   sudo yum install redis
   
   # Arch Linux
   sudo pacman -S redis
   ```

2. **配置 Redis**
   ```bash
   # 编辑配置文件
   sudo nano /etc/redis/redis.conf
   
   # 主要配置项
   port 6379
   bind 127.0.0.1
   requirepass your_redis_password
   maxmemory 256mb
   maxmemory-policy allkeys-lru
   ```

3. **启动服务**
   ```bash
   sudo systemctl start redis
   sudo systemctl enable redis
   
   # 验证状态
   sudo systemctl status redis
   ```

## Maven 安装配置

### Windows 安装步骤

1. **下载 Maven**
   - 访问 Apache Maven 官网
   - 下载 Binary zip archive

2. **安装 Maven**
   ```cmd
   # 解压到 C:\apache-maven-3.9.6\
   # 设置环境变量
   setx MAVEN_HOME "C:\apache-maven-3.9.6"
   setx PATH "%PATH%;%MAVEN_HOME%\bin"
   ```

3. **配置本地仓库**
   ```cmd
   # 创建本地仓库目录
   mkdir C:\maven-repo
   
   # 配置 settings.xml
   <settings>
     <localRepository>C:\maven-repo</localRepository>
     <mirrors>
       <mirror>
         <id>aliyun</id>
         <mirrorOf>central</mirrorOf>
         <url>https://maven.aliyun.com/repository/public</url>
       </mirror>
     </mirrors>
   </settings>
   ```

4. **验证安装**
   ```cmd
   mvn -version
   ```

### Linux 安装步骤

1. **使用包管理器安装**
   ```bash
   # Ubuntu/Debian
   sudo apt install maven
   
   # CentOS/RHEL/Fedora
   sudo yum install maven
   
   # Arch Linux
   sudo pacman -S maven
   ```

2. **手动安装（可选）**
   ```bash
   # 下载并解压
   wget https://downloads.apache.org/maven/maven-3.9.6/binaries/apache-maven-3.9.6-bin.tar.gz
   tar -xzf apache-maven-3.9.6-bin.tar.gz
   sudo mv apache-maven-3.9.6 /opt/
   
   # 配置环境变量
   export M2_HOME=/opt/apache-maven-3.9.6
   export PATH=$PATH:$M2_HOME/bin
   ```

### 数据库表结构概览

项目包含以下核心表：

1. **用户相关表**
   - `sys_user` - 用户信息表
   - `sys_role` - 角色表
   - `sys_user_role` - 用户角色关联表

2. **权限管理表**
   - `sys_menu` - 菜单权限表
   - `sys_role_menu` - 角色菜单关联表

3. **系统管理表**
   - `sys_dept` - 部门表
   - `sys_dict_type` - 字典类型表
   - `sys_dict_data` - 字典数据表

4. **日志审计表**
   - `sys_oper_log` - 操作日志表
   - `sys_logininfor` - 登录日志表

5. **WMS 业务表**
   - `wms_warehouse` - 仓库表
   - `wms_product` - 商品表
   - `wms_product_supplier` - 商品供应商关联表
   - `wms_product_inventory` - 商品库存表

6. **电商模块表**
   - `ecommerce_cart` - 购物车表
   - `ecommerce_order` - 订单主表
   - `ecommerce_order_item` - 订单明细表

## 项目启动

1. **启动数据库和缓存**
   ```cmd
   # 启动 MySQL
   net start mysql
   
   # 启动 Redis
   redis-server.exe --service-start
   ```

2. **启动后端服务**
   ```cmd
   # 进入后端目录
   cd task-manager-backend
   
   # 方式1：使用 Maven
   ./mvnw.cmd spring-boot:run
   
   # 方式2：使用 Java
   java -jar target\task-manager-backend-1.0.0.jar
   
   # 方式3：使用批处理
   start.bat
   ```

3. **启动前端服务**
   ```cmd
   # 进入前端目录
   cd ../task-manager-frontend
   
   # 启动开发服务器
   npm run dev
   ```

### 启动脚本

#### start.bat 脚本内容

```batch
@echo off
chcp 65001 >nul 2>&1
echo ========================================
echo   Task Manager 项目启动脚本
echo ========================================
echo.

:: 启动后端（Spring Boot）
echo [1/2] 启动后端 Spring Boot (端口 8080) ...
start "TaskManager-Backend" cmd /c "cd /d %~dp0task-manager-backend && java -jar target\task-manager-backend-1.0.0.jar"

:: 等待 2 秒
timeout /t 2 /nobreak >nul

:: 启动前端（Vite Dev Server）
echo [2/2] 启动前端 Vite Dev Server (端口 3000) ...
start "TaskManager-Frontend" cmd /c "cd /d %~dp0task-manager-frontend && npm run dev"

echo.
echo ========================================
echo   启动完成！
echo   后端: http://localhost:8080
echo   前端: http://localhost:3000
echo   API文档: http://localhost:8080/doc.html
echo ========================================
pause
```

   ```cmd

   net start mysql
   ```
